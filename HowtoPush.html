Here are the steps to push your code to your own GitHub repository. 

Git bash:

1. Make sure you're in the right directory:

pwd <-- Print Working Directory

2. Make sure you configure your credentials

git config --global user.email "MY_NAME@example.com"

3. Add your remote repository in your local machine

git remote add (localComputer) (url to your repository)

4. Make sure you're in the correct branch

git branch -M main

5. Check the your file using git status

git status 

6. Add all your code to the add phase

git add -A .

7. Then commit your code to the repository

git commit -m "Commit code" 

8. Push your code to the repository

git push -u (localComputer) (currentBranch)
